## APP CRIPTOMONEDA

### Descripción general

Esta aplicación permite consultar el valor actual de distinta criptomonedas.

### Instrucciones

Clonar proyecto

```bash
git clone https://github.com/DnarvaezCode/AG-DEMO-CRIPTO.git
```

Ejecutar comando

```bash
npm i
```

Correr proyecto

```bash
npm start
```
